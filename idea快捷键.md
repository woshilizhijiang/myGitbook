

## idea快捷键总结

### 1.windows模式

```markdown
1.file structure(查找文件方法)
Ctrl+F12

2.change font size with Ctrl+Mouse Wheel
位置：Setting->Editor->General

3.接口方法和实现切换
Ctrl+Alt+B 	:接口方法到实现类方法
Ctrl+U		:实现类方法到接口方法

4.缩写构建
System.out.println() ： sout
main()方法：psvm

5.CTRL +  Shift + U 小写字母转大写

6.注释
Ctrl+/ 或 Ctrl+Shift+/  注释（// 或者/*...*/ ）

7.Ctrl+Shift+上下键  上下移动代码 

8.Ctrl+D 复制行 

9.删除行
Ctrl+X 
Ctrl+Y

10.CTRL+P   方法参数提示  

11.多窗口项目间跳动
下一个窗口：Ctrl + Alt + ]
上一个窗口：Ctrl + Alt + [

12.同一工程文件间跳转
Back按键：Ctrl + Alt + 左箭头
Forward按键：Ctrl + Alt + 右箭头



```
