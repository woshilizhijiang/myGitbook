# Groovy指南

### 下载安装(mac 版本)

https://groovy.apache.org/download.html

```groovy
wget https://dl.bintray.com/groovy/maven/apache-groovy-sdk-3.0.1.zip

cd ~
vi .bash_profile
export PATH=$PATH:/Users/lzj11/Downloads/groovy-3.0.1/bin  

source ~/.bash_profile

groovy -v
```
