# 互联网算法

### 1.一致性hash算法

基本概念：一种特殊的哈希算法，在移除或者添加一个服务器时，能够尽可能小地改变已存在的服务请求与处理请求服务器之间的映射关系。

目的：解决简单哈希算法在分布式[哈希表](https://baike.baidu.com/item/哈希表/5981869)( Distributed Hash Table，DHT) 中存在的**动态伸**缩等问题。

条件：平衡型、单调性、分散性

场景：分布式系统

hash值取值范围：0 - 2^32-1
